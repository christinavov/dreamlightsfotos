# Фото для портфолио

В каждой папке — одна категория съёмок из раздела "Портфолио" на сайте:

- `portrait/` — Порträts
- `love/` — Love Story
- `family/` — Familie
- `fashion/` — Fashion
- `art/` — Art
- `wedding/` — Hochzeiten
- `event/` — Events
- `landscape/` — Landschaft & Natur
- `maternity/` — Schwangerschaft
- `business/` — Business

## Как добавить фото

Просто загрузите файл (`.jpg`, `.jpeg`, `.png` или `.webp`) в нужную папку — например, `images/portfolio/wedding/hochzeit-01.jpg` — и закоммитьте/запушьте изменение (или загрузите через веб-интерфейс GitHub кнопкой "Add file").

После этого автоматически запустится GitHub Action, который пересоберёт список фото (`images/portfolio/manifest.json`), и в течение минуты новые фото появятся на сайте в разделе "Портфолио" — без правки кода.

Если в папке категории нет ни одного фото, на сайте вместо неё показывается временная градиентная заглушка.

## Важно

- Название файла не влияет на подпись — подписью служит название категории.
- Если фото не появилось на сайте дольше нескольких минут после загрузки, проверьте вкладку **Actions** в репозитории на GitHub — там будет видно, отработал ли workflow `Update gallery manifest` и не было ли ошибки.
