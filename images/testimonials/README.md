# Скриншоты отзывов

Загрузите сюда скриншоты отзывов клиентов (переписка в WhatsApp, Instagram, Google-отзывы и т.д.) в формате `.jpg`, `.jpeg`, `.png` или `.webp`.

После загрузки и коммита GitHub Action `Update gallery manifest` автоматически пересоберёт `images/testimonials/manifest.json`, и в течение минуты скриншот появится на сайте в разделе "Отзывы" — рядом с существующими отзывами, без правки кода.

Название файла не важно, порядок показа — по имени файла (можно называть `01-anna.jpg`, `02-maria.jpg` и т.п., чтобы управлять порядком).
