# Фото для блока "Обо мне"

Положите сюда фотографию и назовите файл `photo.jpg` (или `photo.png` — тогда поправьте расширение в `index.html`, строка с `<img src="images/about-me/photo.jpg" ...>`).

Сайт автоматически подхватит фото на странице в разделе "Обо мне" — ничего больше менять не нужно. Пока файла нет, вместо фото показывается градиентная заглушка.

Рекомендации по фото:
- Портретная ориентация, соотношение сторон примерно 4:5 (например, 1200×1500 px)
- Хорошо освещённое, крупный план
