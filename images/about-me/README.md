# Фото для блока "Обо мне"

Фотографии из этой папки показываются в разделе "Обо мне" на сайте и автоматически сменяют друг друга каждые 2 минуты (с плавным затуханием).

Список файлов задан в `js/script.js` в массиве `aboutPhotos` (в начале файла):

```js
const aboutPhotos = ["images/about-me/IMG_1588.JPG", "images/about-me/IMG_1589.JPG"];
```

Чтобы добавить, убрать или заменить фото — положите/удалите файл в этой папке и обновите список `aboutPhotos` в `js/script.js` (порядок в массиве = порядок показа).

Рекомендации по фото:
- Портретная ориентация, соотношение сторон примерно 4:5
- Хорошо освещённое, крупный план
