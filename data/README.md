# Опубликованные текстовые отзывы

`reviews.json` — список отзывов, которые показываются на сайте в разделе "Отзывы" (под скриншотами). Формат — простой список объектов:

```json
[
  { "name": "Anna K.", "text": "Das Shooting war wunderschön, danke!" },
  { "name": "Marco B.", "text": "Sehr professionell und entspannt." }
]
```

## Как это работает

1. Посетитель сайта заполняет форму "Hinterlasse eine Bewertung" под отзывами.
2. При отправке открывается почтовый клиент с письмом на `dreamlightfotos@hotmail.com`, куда уже вставлены имя и текст отзыва — отзыв **не публикуется автоматически**, это сделано специально, чтобы на сайт не мог кто угодно опубликовать спам или оскорбления без вашей проверки.
3. Прочитав письмо, вы решаете, публиковать отзыв или нет.

## Как опубликовать отзыв

Откройте `data/reviews.json` в GitHub (кнопка "Edit" — карандаш) и добавьте новый объект в список:

```json
{ "name": "Имя из письма", "text": "Текст отзыва из письма" }
```

Не забудьте запятую после предыдущего объекта, если список не пустой. Закоммитьте изменение — отзыв появится на сайте в течение минуты.

## Как удалить отзыв

Откройте `data/reviews.json`, найдите нужный объект `{ "name": ..., "text": ... }` и удалите его целиком (вместе с фигурными скобками и, если нужно, лишней запятой). Закоммитьте — отзыв исчезнет с сайта.
