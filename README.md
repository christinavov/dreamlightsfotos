# Dream Lights Photos

Сайт-визитка фотографа в чёрном стиле с градиентными акцентами. Статический HTML/CSS/JS, без сборки и зависимостей.

Многоязычный сайт: немецкий (основной язык) и английский (второй), с переключателем языка (флаг + название) в шапке.

## Структура

- `index.html` — разметка страницы (Hero, Обо мне, Портфолио, Услуги, Отзывы, Контакты)
- `css/style.css` — стили (тёмная тема, градиенты, адаптивная вёрстка, переключатель языка)
- `js/script.js` — интерактив: мобильное меню, переключатель языка (DE/EN) с сохранением выбора в localStorage, фильтры портфолио, лайтбокс, форма, анимации при скролле

## Запуск локально

Открыть `index.html` в браузере, либо запустить локальный сервер:

```bash
python3 -m http.server 8000
```

и перейти на `http://localhost:8000`.

## Заметки

- Фотографии в портфолио и в блоке "Обо мне" — плейсхолдеры на градиентах (реальных фото нет). Замените `.gallery-item__ph` / `.about__photo` фоновые градиенты на реальные изображения через `background-image` или тег `<img>`.
- Форма обратной связи работает в демо-режиме (без бэкенда) — при отправке просто показывает сообщение об успехе.
- Переводы хранятся в объекте `translations` в `js/script.js` (ключи вида `hero.title1`). Чтобы добавить текст, добавьте элемент с атрибутом `data-i18n="ключ"` (или `data-i18n-placeholder` / `data-i18n-aria` для плейсхолдеров и aria-label) в `index.html` и соответствующие переводы для `de` и `en`.
- Контакты (телефон, email, город) и цены на услуги — примерные, замените на реальные данные.
