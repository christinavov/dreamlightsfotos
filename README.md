# Dream Lights Photos

Сайт-визитка фотографа в чёрном стиле с градиентными акцентами. Статический HTML/CSS/JS, без сборки и зависимостей.

Многоязычный сайт: немецкий (основной язык) и английский (второй), с переключателем языка (флаг + название) в шапке.

## Структура

- `index.html` — разметка страницы (Hero, Обо мне, Портфолио, Услуги, Отзывы, Контакты)
- `css/style.css` — стили (тёмная тема, градиенты, адаптивная вёрстка, переключатель языка)
- `js/script.js` — интерактив: мобильное меню, переключатель языка (DE/EN) с сохранением выбора в localStorage, фильтры портфолио, лайтбокс, форма, анимации при скролле

## Запуск локально

Открыть `index.html` в браузере, либо запустить локальный сервер:

```bash
python3 -m http.server 8000
```

и перейти на `http://localhost:8000`.

## Портфолио: как загрузить фото

Фото для галереи портфолио лежат в `images/portfolio/<категория>/` (например, `images/portfolio/wedding/`). Загрузите файл `.jpg`/`.jpeg`/`.png`/`.webp` в нужную папку — GitHub Action `Update gallery manifest` (`.github/workflows/update-gallery-manifest.yml`) автоматически пересоберёт `images/portfolio/manifest.json`, и в течение минуты фото появится на сайте без правки кода. Пока в папке категории нет фото — на сайте показывается временная градиентная заглушка. Подробности — в `images/portfolio/README.md`.

Фото для блока "Обо мне" — по такому же принципу, см. `images/about-me/README.md` (там ротация фото задаётся вручную в `js/script.js`, автоманифеста нет).

## Отзывы: как загрузить скриншот

Скриншоты отзывов лежат в `images/testimonials/`. Загрузите файл `.jpg`/`.jpeg`/`.png`/`.webp` — тот же GitHub Action пересоберёт `images/testimonials/manifest.json`, и скриншот автоматически появится в разделе "Отзывы" рядом с существующими карточками. Подробности — в `images/testimonials/README.md`.

## Отзывы: форма для посетителей

Под отзывами на сайте есть форма "Hinterlasse eine Bewertung" — посетитель вводит имя и текст, при отправке открывается его почтовый клиент с письмом на `hello@dreamlightsfotos.com`. Отзыв **не публикуется автоматически** (чтобы на сайт нельзя было накидать спам без проверки) — чтобы опубликовать его, добавьте запись в `data/reviews.json`. Чтобы убрать отзыв — удалите соответствующую запись из того же файла. Подробная инструкция с примерами — в `data/README.md`.

## Заметки

- Форма обратной связи работает в демо-режиме (без бэкенда) — при отправке просто показывает сообщение об успехе.
- Переводы хранятся в объекте `translations` в `js/script.js` (ключи вида `hero.title1`). Чтобы добавить текст, добавьте элемент с атрибутом `data-i18n="ключ"` (или `data-i18n-placeholder` / `data-i18n-aria` для плейсхолдеров и aria-label) в `index.html` и соответствующие переводы для `de` и `en`.
- Контакты (телефон, email, город) и цены на услуги — примерные, замените на реальные данные.
